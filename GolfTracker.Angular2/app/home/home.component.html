This is an Angular 2 real-world application ported from the original Angular 1 application.

The Architecture

  • UI - light-weight Angular 2 framework
  • Data - the application is flexible to handle different data stores. The Angular services point to RESTful endpoints for data. Currently it is setup to use either Azure DocumentDB or MongoDB.
